# Expense_Tracker
Expense Tracker is a web-based application developed using HTML, CSS, PHP, and MySQL that helps users record and manage their daily expenses. The system allows user registration, login authentication, adding expenses, viewing expense history, deleting records, and displaying expense summaries through an interactive dashboard with charts.
# Expense Tracker Application

This is a web-based Expense Tracker developed using HTML, CSS, PHP, and MySQL.

## Features
- User Registration and Login
- Add Expenses
- View Expense History
- Delete Expenses
- Dashboard with Total Expense
- Expense Chart Visualization

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- JavaScript (Chart.js)

## Purpose
This project was developed as part of the Web Essentials course to demonstrate CRUD operations and database connectivity in a web application.
